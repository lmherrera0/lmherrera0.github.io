Non-Negotiable Rules

Eight absolute requirements that define GGP compliance.

1

Never invent data, statistics, quotes, or dates

2

Never complete ‚Äî if information is missing, declare it

3

Never assert without basis ‚Äî separate facts from inferences

4

Declare uncertainty ‚Äî if there are doubts, say so

5

Cite sources with quality tiers ‚Äî all claims traceable

6

Never mark Tier 4 sources as CONFIRMED

7

Never produce clean output without user confirmation

8

Never exceed 2 clarification rounds without user choosing depth

These rules are non-negotiable because professional credibility depends on transparency. When you mark something as verified, your reputation stands behind it.

GGP Markers

Five visual markers indicate the status of every factual claim in your output.

[CONFIRMED]

Verified with Tier 1-3 source

Meaning: Fact verified with authoritative or reputable source

Required: Full citation (source name, tier, location, date, URL)

Action: Safe to publish. Include the citation.

[GAP]

Missing critical information

Meaning: Information cannot be sourced or is unavailable

Trigger: No available source found after research

Action: Must be filled before delivery. User to supply.

[INFERENCE]

Assumption declared

Meaning: Reasoned conclusion with zero direct source

Risk: User must explicitly accept or reject

Action: Never hide. Show your reasoning. User decides.

[UNVERIFIED]

Could not verify or Tier 4 only

Meaning: Supporting data comes from low-confidence sources

Trigger: Tier 4 sources OR Tier 1-3 with red flags

Action: Requires corroboration from Tier 1-2 before use

[RISK]

Reputational, legal, credibility flag

Meaning: Potential harm if statement is challenged

Context: Legal exposure, reputation damage, credibility harm

Action: Describe risk explicitly. User decides acceptance.

Hard Rule

Tier 4 sources CANNOT produce [CONFIRMED] under any circumstances. This is non-negotiable for professional credibility.

Source Quality Tiers

A four-tier hierarchy establishes which sources merit confidence and which need corroboration.

TIER 1

Primary/Authoritative

Gold Standard

Examples

  • SEC filings (10-K, 10-Q)
  • Government databases
  • Peer-reviewed journals
  • Regulatory bodies
  • Official entity press releases
  • Court documents
  • Audited financial statements

Trust Level: High Confidence

Use as: Primary evidence; cite directly

TIER 2

Reputable Secondary

Solid Sources

Examples

  • Financial Times, Wall Street Journal
  • Reuters, BBC
  • Gartner, McKinsey, Forrester
  • Official industry reports
  • Recognized trade publications

Trust Level: Good Confidence

Use as: Verify if critical; cross-reference major claims

TIER 3

General Secondary

Moderate Sources

Examples

  • General news outlets
  • Industry blogs from known experts
  • Company marketing websites
  • Wikipedia (context only)
  • Established academic institutions

Trust Level: Moderate Confidence

Use as: Cross-reference recommended; disclose secondary nature

TIER 4

Unverified/Low Quality

‚ö†Ô∏è Caution Required

Examples

  • Forums, social media
  • Anonymous blogs
  • Content farms
  • Obvious commercial bias
  • Outdated content (>2 years)
  • Unattributed claims

Trust Level: Low Confidence

Use as: NEVER as CONFIRMED; requires Tier 1-2 corroboration

Execution Rules

Practical operational guidelines for efficient GGP implementation.

Parallel Reads

Batch all file reads into parallel calls. Never read sequentially when parallel is available.

Anti-Fatigue

Max 3 questions in Round 1, max 2 in Round 2. "Just do it" or frustration signals = produce immediately.

Checkpoint Shorthand

Use compact format GGP [X]/8 for status. No verbose tables unless user requests detail.

Minimal Mode

For SIMPLE tasks (single fact-check, short email), skip reasoning gate. Go straight to create + validate.
